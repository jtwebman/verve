# Verve Language for VS Code / Cursor

Syntax highlighting for `.vv` files.

## Install (local)

1. Copy or symlink this folder to your VS Code extensions directory:
   ```
   ln -s /path/to/verve/editors/vscode ~/.vscode/extensions/verve-lang
   ```
2. Restart VS Code / Cursor
3. Open any `.vv` file

## Features

- Syntax highlighting for all Verve keywords, types, and constructs
- Auto-closing brackets and quotes
- Comment toggling with `Ctrl+/`
- Indentation support
